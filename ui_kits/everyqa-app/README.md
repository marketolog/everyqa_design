# UI-кит · Приложение EveryQA

Интерактивная сборка ключевых экранов будущего приложения (десктоп 1280, светлая + тёмная тема). Открыть: `index.html`.

Экраны (переключаются левой навигацией; онбординг — по «Выйти»):
- **OnboardingScreen** — вход: уровень, email, старт бесплатно
- **DashboardScreen** — «До первого оффера», продолжить обучение, карта развития, ачивки
- **LessonScreen** — урок: видео, конспект, табы Теория/Практика/Тест
- **PracticeScreen** — тренажёр баг-репортов: форма → таблица, тост «Эврика!»
- **ChatScreen** — чат с Архимедом (канированные ответы)
- **PricingScreen** — 3 тарифа, выбор, диалог оплаты
- **ProfileScreen** — профиль, навыки, настройки, сброс прогресса

Все экраны собраны из компонентов `components/` — ничего не переопределяют. Данные фиктивные.
